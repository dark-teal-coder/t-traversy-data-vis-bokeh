# Steps

- install python
- access to `pip` tool
- `pip install pipenv`
- access to `pipenv` command
- run `pipenv shell`
- `Pipfile` file created
- `pipenv install bokeh pandas`
- install bokeh and pandas packages in the virtual env
- `Pipfile.lock` file created
- `python main.py` to generate the HTML output file
